MENSAGENS QUE O RMI SERVER RECEBE:

Registo aceite:
type | registerComplete ; username | *o username*

Registo com username existente:
type | invalidRegister ; username | *username invalido*

Login valido:
type | validLogin ; username | *username*

Login invalid:
type | invalidLogin

Resultado de pesquisa:
type | searchResult ; urlCount | *numero de urls* ; url_0 | *1º url* ; url_1 | *2º url* ;etc.

Receber historico de pesquisa de um utilizador
type | searchHistoryResults ; searchCount | *numero de pesquisas* ; word_0 | *1ª pesquisa mais recente* ; etc.







MENSAGENS QUE O RMI SERVER MANDA:

Registar:
type | register ; username | *username* ; password | *password*

Login:
type | login ; username | *username* ; password | *password*

Indexar um novo url a uma palavra
type | index ; word | *palavra* ; url | *url associado*

Pesquisar uma palavra
type | serach ; word | *palavra*

Pesquisar uma palavra por utilizador logado
type | serach ; word | *palavra* ; username | *username*

Historico de pesquisas de um utilizador
type | searchHistory ; username | *username*
